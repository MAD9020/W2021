# Week 5 In-class Exercise: Sketching and iterating wireframes

# Details

# Brainstorm by sketching wireframes (2x15 mins)

Using what you learned in last week's lesson about navigation models, and what you learned this week about wireframing, sketch out some concepts for how you could display the information in an app (Doors Open Ottawa).

  1. Sketch 3 concepts, each using a different navigation model for:

    • The main page/landing page where the app would open
    • How users will access the main sections in your information architecture

  2. Share with a classmate, and get their feedback on it.

  3. Take the feedback, and reiterate. Sketch out:

    • The main page/landing page where the app would open
    • How users will access the main sections in your information architecture
    • What users will see when they select one of those main sections
    • How they would view/interact with content in that section (sub-navigation?)
    • How they would move back to the first screen.

# Deliverable for this exercise:

  • Photo/scan of the two rounds of sketching/wireframing
  • Save them as ONE PDF file and ensure the orientation is correct for comments
  • Filename: firstNameLastName_Wireframe.pdf

Note

  • Use only pen/paper for your sketching. No digital tools yet!
  • Access paper wireframe templates in Week 05 module
  • Install Adobe Acrobat so that you can assemble all screens into one PDF file.
